A dynamic WordPress theme for movie uploading and streaming websites with advanced features and modern design.

== Description ==

MovieFlix is a comprehensive WordPress theme designed specifically for movie streaming and download websites. Built with modern web standards and best practices, it offers a Netflix-inspired dark design with powerful functionality.

= Key Features =

* **Custom Movie Post Type** - Dedicated post type for movies with detailed metadata
* **Advanced Taxonomies** - Organize movies by genres, languages, quality, and release years
* **Dynamic Search** - Real-time AJAX search with autocomplete functionality
* **Advanced Filtering** - Filter movies by multiple criteria simultaneously
* **Comments System** - Custom movie comments with reply functionality
* **Contact Management** - Built-in contact form with database storage
* **Download Tracking** - Track download statistics by quality
* **View Tracking** - Monitor movie popularity and engagement
* **Social Sharing** - Share movies on social platforms
* **SEO Optimized** - Schema markup and meta tags for better search visibility
* **Admin Dashboard** - Comprehensive management interface

= Design Features =

* **Fully Responsive** - Optimized for all devices with mobile-first approach
* **Dark Theme** - Netflix-inspired design with customizable colors
* **Lazy Loading** - Improved performance with image lazy loading
* **Smooth Animations** - CSS transitions and hover effects
* **Accessibility** - WCAG compliant with proper ARIA labels
* **Progressive Enhancement** - Works without JavaScript

= Technical Features =

* **AJAX Powered** - Seamless user experience without page reloads
* **Security** - Nonce verification and input sanitization
* **Performance** - Optimized queries and caching support
* **Customizable** - WordPress Customizer integration
* **Translation Ready** - Internationalization support

== Installation ==

1. Upload the theme folder to `/wp-content/themes/`
2. Activate the theme through the 'Appearance > Themes' menu in WordPress
3. Go to 'Appearance > Customize' to configure theme options
4. Set up your site logo, colors, and preferences
5. Configure dynamic pages (About, Contact, Privacy Policy)

== Frequently Asked Questions ==

= How do I add movies? =

Go to Movies > Add New Movie in your WordPress admin. Fill in the movie details, upload a poster image, and add download/watch links.

= Can I customize the colors? =

Yes, go to Appearance > Customize > MovieFlix Theme Options to change the primary color and other settings.

= How do I manage contact messages? =

Contact messages are stored in the database and can be managed through MovieFlix > Contact Messages in the admin.

= Is the theme SEO friendly? =

Yes, the theme includes Schema.org markup, proper meta tags, and follows SEO best practices.

= Does it work with caching plugins? =

Yes, the theme is compatible with popular caching plugins like WP Rocket and W3 Total Cache.
